# 🎓 Scholarship Fund Tracker

A full-stack application built to learn backend integration, database management, containerization, and cloud deployment workflows.

## 📌 Project Objective

The primary goal of this project was not frontend development, but to gain hands-on experience in:

* Building and integrating APIs
* Working with databases
* Managing application architecture
* Learning Docker containerization
* Preparing applications for AWS deployment
* Understanding real-world DevOps workflows

---

## 🚀 Features

### Authentication

* User Signup
* User Login
* Password Hashing using Passlib

### Scholarship Management

* Add Scholarship Records
* Edit Scholarship Records
* Delete Scholarship Records
* Search Scholarships

### Expense Management

* Add Expense Records
* Edit Expense Records
* Delete Expense Records
* Search Expenses

### Financial Dashboard

* Total Scholarships
* Total Expenses
* Current Savings
* Goal Tracking
* Financial Analytics Visualization

### Goal Tracking

* Savings Goal Progress
* Percentage Completion
* Remaining Amount Calculation

## ⚙️ Local Setup

### Clone Repository

```bash
git clone https://github.com/M-g-cell/ScholarshipFundTracker.git
cd ScholarshipFundTracker
```

### Backend

```bash
cd backend

python -m venv venv

venv\Scripts\activate

pip install -r requirements.txt

uvicorn main:app --reload
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```
